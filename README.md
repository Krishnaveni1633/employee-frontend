# 🎨 Employee Management System — Frontend

React frontend for the Employee Management System API.

## 🌍 Live Demo
- **Frontend:** https://employee-frontend-zjq0.onrender.com
- **Backend API:** https://employee-management-api-59jd.onrender.com/docs

## 📌 About
A React frontend that connects to the Employee Management System API with login, dashboard, and employee management features.

## 🚀 Tech Stack
- **React.js** — User interface
- **Axios** — API calls
- **React Router** — Navigation
- **React Toastify** — Notifications

## ✅ Features
- Login page with JWT authentication
- Dashboard with welcome message and role display
- Employee list with search and filter
- Add new employee form
- Delete employee
- Protected routes
- Logout

## 🛠️ How to Run Locally

```bash
git clone https://github.com/Krishnaveni1633/employee-frontend.git
cd employee-frontend
npm install
npm start
```

Open `http://localhost:3000`
